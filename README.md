# Portfolio

My personal portfolio site. Projects, write-ups, and links, built as a single static page.

**Live:** [prxmrrr.github.io/portfolio](https://prxmrrr.github.io/portfolio/)

## What's here

A hero landing section, a project showcase with detail overlays for each build, an image gallery, and a contact section. The featured projects are Tekkendle, ShopWatch, Kōkan, and Aricstoll, each with its own write-up covering what it does and how it's built.

## How it's built

One HTML file, no framework. Vanilla JavaScript handles the project overlays, scroll reveals, the rotating hero word, and a light/dark theme toggle that remembers your choice. Gallery images are inlined as base64, so the page is self-contained with no external asset requests.

## Stack

HTML, CSS, vanilla JavaScript. Deployed on GitHub Pages.

## Running it locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server
```
